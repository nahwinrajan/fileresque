Recovery utility for macOS

A calm hand when a file goes missing.

FileResque scans your disk for deleted files, tells you honestly how likely each one is to come back, and restores it safely to another drive. No account, no upload, no false hope.

↓ Download for macOS View source on GitHub

Universal build · macOS 12 Monterey or later · Apple Silicon & Intel

First launch: this is a free, unsigned build — on the first open, right-click the app → Open (or System Settings → Privacy & Security → Open Anyway). One time only.

Built for the moment a file goes missing

Calm competence over panic: find the drive, show what's recoverable, and get out of your way.

🎯

Honest recovery odds

Every file is rated High, Medium, or Low before you commit, based on whether its data blocks are still intact on disk.

🔒

Never touches the network

FileResque makes zero network calls. Your file names and data stay on your machine — there's nothing to upload and nowhere to leak.

🧭

Safe by design

Recovers only to a different drive, so it never overwrites the data it's rescuing, and verifies every file with a SHA-256 digest.

🗂️

APFS & HFS+

Reads Apple's modern and legacy filesystems directly at the block level — internal drives, external SSDs, and USB disks.

⚡

Light on your Mac

Built in Rust. Streams files block by block, so recovering a 10 GB video uses kilobytes of memory, not gigabytes.

💛

Free and open source

GPLv3 — no licence key, no trial wall, no upsell. Read the code, build it yourself, or send a patch.

Three steps

From "oh no" to "got it back" without a manual.

Pick a disk

Grant Full Disk Access once, choose the drive the file was on, and start a scan.

Check the odds

Click any deleted file to see its recovery probability and how much is still readable.

Recover it

Point FileResque at another drive and it writes the file back, verified byte-for-byte.

Your data never leaves your Mac

Recovery software asks for the keys to your whole disk. FileResque earns that trust by staying completely offline.

// Zero network. Ever.

Before you scan: if a file really matters, stop using the drive it was on right away. Every time the disk writes new data, it can land on top of what you're trying to recover. FileResque can't promise a file back — deletion, TRIM on SSDs, and reuse all work against recovery — but it will tell you the truth about your odds instead of selling you false hope.
